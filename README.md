# PresetGold

PresetGold es una extensión para facilitar el flujo de trabajo con presets.

## Funciones
- Búsqueda rápida de presets y aplicación directa.
- Atajos de teclado para un flujo de trabajo más ágil.
- Interfaz integrada en la aplicación.

## Cosas por arreglar (Problemas conocidos)
- **Error en custom presets que involucren color:** No se eligen los colores correctos.
- **Capas de Transformar:** Se añaden nuevas capas de Transformar y los keyframes se importan a una misma capa de Transformar.
